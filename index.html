Hawaii Map Editor

Send data to your Google Sheet in the required format for the Hawaii game.
Fill only the fields you need — empty fields are OK for most cases.

Level & Map Image Required — whole number only Path to the map image on GitHub. Leave empty if not changing background.
Zoom Region (optional) Space-separated x,y pairs for the clickable polygon area. Level to zoom to when the region is clicked. Leave empty if not a zoom area.
Interactive Object (optional) Select if adding a marker or object. Leave "— None —" if not.
Single x,y coordinate for the object position. JSON string for properties (name, desc, image, etc.). Leave empty if not needed.
