Hawaii Map Editor

Add new map levels, zoom regions and interactive objects.
Everything is saved to your Google Sheet — changes appear in the live map after refresh.

1. Level & Main Map Image Required — whole number only Only required when: creating a new level or changing the background image.
Leave empty when: adding markers/objects/NPCs to an existing level — the map will keep its current background.
Quick steps for new images:
  1. Select file above → preview appears + suggested path shown
  2. Open your GitHub repo in another tab
  3. "Add file" → "Upload files" → drag/drop image
  4. Place in images/ folder → commit
  5. Copy suggested path → paste here (only if creating/changing the map background)

Paths are case-sensitive! Use lowercase letters and dashes (-), no spaces.

2. Zoom Region (optional)

Define a clickable polygon area that zooms to another level when tapped/clicked.

Format: space-separated x,y pairs (clockwise or counterclockwise).
Recommended tool: image-map.net → upload your map image → click points → copy the coordinates Leave empty if this is not a zoom trigger area
3. Interactive Object / Marker (optional)

Add something clickable on the map (marker, icon, NPC, item…)

Single coordinate — center of the object/marker
Object image steps:
  1. Select file above → see preview
  2. Upload to GitHub → images/icons/ or images/npcs/ folder recommended
  3. Copy suggested path into the next field
